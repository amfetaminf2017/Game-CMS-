*************
* Установка *
************* 
1. Распаковать архив у на хостинге. 
2. Импортировать бд из файла game.sql. 
3. Прописать данные для подключения к бд по пути /core/function.php
4. Зарегистрировать игрока. 
5. В базе данных дать своему игроку access -> 2.
